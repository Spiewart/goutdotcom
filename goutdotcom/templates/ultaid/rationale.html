Organ transplants make everything hard. If you have had one, you absolutely need to discuss your gout management with a rheumatologist.

Allopurinol Hypersensitivity: {% if object.allopurinol_hypersensitivity == True %} present {% elif object.allopurinol_hypersensitivity == False %} absent {% else %} not reported {% endif %}

Allopurinol is generally well tolerated, but it does have a rare and serious side effect known as allopurinol hypersensitivity syndrome. If you have had this, or had a drug rash or other side effect, from allopurinol in the past, you need to discuss your gout care with a rheumatologist. Historically, patients who needed allopurinol but had a history of side effects were de-sensitized. In the modern era, we just use febuxostat instead.

Febuxostat Hypersensitivity: {% if object.febuxostat_hypersensitivity == True %} present {% elif object.febuxostat_hypersensitivity == False %} absent {% else %} not reported {% endif %}

Similar to allopurinol, some patients have reactions to febuxostat that preclude its use. It is uncommon to start a patient on febuxostat before allopurinol, so if you aren't able to tolerate both allopurinol and febuxostat your gout care will need to be discussed with a rheumatologist.

Cardiac History: {% if object.MACE == True %} present {% elif object.MACE == False %} absent {% else %} unknown {% endif %}

Large clinical trials of febuxostat were associated with an increased risk of serious cardiovascular events, such as heart attacks, as well as mortality due to cardiac causes. Whether this is a true side effect is hotly debated. If you are prescribed febuxostat and have a history of heart disease, you should discuss the risks and benefits with your doctor.
