ULT Conditionally Indicated

Conditional Recommendation

Urate-lowering therapy (ULT) is conditionally recommended for your gout.

This means there is some general medical consensus that you would benefit from ULT, but it is not a "strong" recommendation. It also means that, generally, there is less scientific evidence that you will benefit. It is more a matter of expert opinion.

It's still a personal choice and you need to weigh the pros and cons of the decision for yourself.

Rationale

Number of Flares: {{ object.num_flares }}

{% if object.num_flares == 'zero' %}

Not ever having a gout flare is a strong recommendation against using urate-lowering therapy.

{% elif object.num_flares == 'one' %}

ULT is generally not recommended for an individual's first gout flare. This is because the time that passes between flares will sometimes be as long as years and it is not necessarily worth it to take a medication daily to prevent infrequent attacks.

{% else %} {% if object.freq_flares %} {% if object.freq_flares == "one" %}

If you've had more than one flare, the decision to treat is generally based on how frequently you are getting them.

{% elif object.freq_flares == "Two or more" %}

If you are having more than one gout flare per year, ULT is strongly recommended. This is to prevent the pain and disability of recurrent flares. Also, individuals who get more than one flare per year are more likely to have their gout accelerate, becoming more frequent, disabling, and causing long-term joint damage.

{% endif %} {% endif %} {% endif %}

Frequency of Flares: {{ object.freq_flares }} / year

{% if object.freq_flares == 'one' %}

Generally, only having 1 flare per year isn't burdensome enough to warrant daily medication on an indefinite basis. There are, however, exceptions to this, such as when you have erosions, tophi, hyperuricemia, CKD, or urate kidney stones.

{% elif object.freq_flares == 'two or more' %}

Having more than one gout flare per year is a strong indication for ULT.

{% endif %}

Erosions: {% if object.erosions == True %} present {% elif object.erosions == False %} absent {% else %} unknown {% endif %}

These are often found on x-rays your doctor performed on the joints that you report having gout in. They can also be seen on CT scans, MRIs, and ultrasounds of your gouty joints.

Erosions are an indication for urate-lowering therapy. This is important for your long-term health because erosions are permanent changes to the bone and cartilage. They can be prevented with ULT, but once present can't be reversed.

Tophi: {% if object.tophi == True %} present {% elif object.tophi == False %} absent {% else %} unknown {% endif %}

Tophi are an indication for ULT because they can cause permanent damage to the bone and cartilage they are adjacent to. They can also break down, drain, and become infected.

Uric Acid Kidney Stones: {% if object.stones == True %} present {% elif object.stones == False %} absent {% else %} unknown {% endif %}

Kidney stones are terrible. If you have uric acid kidney stones and gout, using ULT to get rid of both is a no-brainer (conditional recommendation).

Chronic Kidney Disease: {% if object.ckd == True %} present {% elif object.ckd == False %} absent {% else %} unknown {% endif %}

Chronic kidney disease (CKD) is associated with incident and progressive gout (2, 3).

Hyperuricemia: {% if object.uric_acid == True %} present {% elif object.uric_acid == False %} absent {% else %} unknown {% endif %}

Hyperuricemia (elevated blood levels of uric acid) is a conditional indication for ULT if you have had a gout flare.
